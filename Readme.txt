This folder contain

2. Report

How to run
1. You have to install: Python, Pip and other needed package in file named install_req.txt
2. Open file Analysis.py
3. Open termianl and type "python(3) Analysis.py"

Where is the result
1. In the report
2. In file ValueCustomer.xlsx
3. In all the images started with "fig"

Data set is from: http://archive.ics.uci.edu/ml/datasets/online+retail
